Project - Database Android App(Doctor's Helper)
Author - Anant Gwari (https://github.com/AnantGwari)
Language - JAVA

The following is an android studio program. The program used android framework provided by google and android studio. The app purposes is to retrieve information from the user in side Activity, parse it into an Intent and pass it down to the Main Activity for display. The app redirects user to the starting screen and show the results.
